Using mission packs by moving them to the "baseq2" folder

Please follow this steps to use mission packs or mods by moving them to the "baseq2" folder:

1 Quit Quake II.
2 Rename the file "PAK0.PAK" within the mission pack folder to "PAK3.PAK".
3 Move the file "PAK3.PAK" from the folder of the mission pack to the "baseq2" folder.
4 Move the file "GameMac.q2plug" from the folder of the mission pack to the "baseq2" folder.
5 Start Quake II.
