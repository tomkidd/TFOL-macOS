Quake II Help
Installation

Please follow this steps to install the MacOS X version of Quake II:

1 Install the Classic MacOS or PC retail version. All install styles are supported (minimum, full, with/without cinematics).
2 Move the content of "Into 'baseq2' folder" (which omes with this package) into the installed "baseq2" folder.
3 Optional: Move the content of the folders called "Into 'xxx' folder" to the approriate folders if you have the "capture the flag", "Rogue", "Xatrix" or "Action Quake II" mission pack installed.
4 Move the application "Quake II" whereever you like.
5 Optional: Move the command-line application "Q2DedicatedServer" whereever you like (preferable to the "Quake II folder". The "Quake II folder" is the folder that holds the "baseq2" folder).

Note: You will be asked to locate the "baseq2" folder which comes with the retail version of Quake II, if you startup Quake II for the first time and have not placed the Quake II application inside your Quake II folder (the "Quake II folder" is the folder that holds the "baseq2" folder).
