Quake II Help
Overview

This is the Cocoa version of the popular game Quake II. It includes the software and OpenGL renderer plug-ins. This package comes with the following files:
• "Into 'baseq2' folder/GameMac.q2plug": The standard game plug-in.
• "Into 'ctf' folder/GameMac.q2plug": The "capture the flag" multiplayer game plug-in.
• "Into 'rogue' folder/GameMac.q2plug": The "Rogue" mission pack game plug-in.
• "Into 'xatrix' folder/GameMac.q2plug": The "Xatrix" mission pack game plug-in.
• "Into 'action' folder/GameMac.q2plug": The "Action Quake II" game plug-in.
• "Quake II": The Quake II application.
• "Q2DedicatedServer": The Quake II server only software. This is a command line application.

and a folder called "qw" (required by the QuakeWorld server software).

Quake II requires MacOS X v10.0 or later and the retail version of Quake II. It doesn't matter if you own the Classic MacOS or the PC version, both will work. Additionally you'll need a graphics board that provides accelerated OpenGL rendering if you want to use the OpenGL renderer instead of the software renderer.
